# gym-algorithmic

This repository contains the algorithmic environments previously present in OpenAI Gym prior to Gym version 0.19.0.

## Environments

Copy-v0  
DuplicatedInput-v0  
RepeatCopy-v0  
Reverse-v0  
ReversedAddition-v0  
ReversedAddition3-v0
